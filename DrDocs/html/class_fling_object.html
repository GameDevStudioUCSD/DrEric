Filename: FlingObject.cs
Author: Michael Gonzalez
Contributing Authors: N/A
Date Drafted: ??? (Sometime during finals Spring Quarter '15)
Description: This script translate specific user mouse movement into an impulse vector that flings the parent object through 2D space. This impulse vector is constructed by recording where the user clicks and releases their mouse.
