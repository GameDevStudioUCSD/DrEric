Detailed Description

Filename: RandomRotations.cs
Author: Michael Gonzalez
Contributing Authors: N/A
Date Drafted: 8/2/2015
Description: This script will wildly shake its parent object. It grabs a reference to the parent's transform and rotates the parent by an origin by a definable range. The origin, or starting rotation, may be shifted by public offsets.
